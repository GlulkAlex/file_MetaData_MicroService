API Basejump: File Metadata (size) Microservice

Objective:

Build a full stack JavaScript app
that is functionally similar to this
reference case
and
deploy it to Heroku.

User story:

  • user can
    submit a FormData object
    that includes a file upload.

  • after submission,
    user will receive
    the file size in bytes
    within the JSON response.

Usage examples (live demo):

  • Link: https://api-file-metadata-microservice.herokuapp.com/
  • Input sample: package.json
  • Output samples: {"file_Size": 475}
